git init : Pour activer un dossier comme repository Git

git add NomDuFichier : permet d'indexer les fichiers à Git

git commit -m "Mon Commentaire" : pour faire un commit avec message

git log : affiche la liste de tous les commits que vous avez réalisés

git commit -a -m "Mon commentaire" ou git commit -am "Mon commentaire" : fusion de la commande git add et git commit

git checkout SHADuCommit: pour se positionner sur un commit donné 

git checkout master: Pour revenir à votre branche principale (au commit le plus récent)

git commit --amend -m "Votre message" : permet de modifier le message du dernier commit

git revert SHADuCommit: Pour "revert" un commit, c'est-à-dire créer un nouveau commit qui fait l'inverse du précédent 

git reset --hard: Pour annuler tous les changements qui n’ont pas encore été commités

git clone lienFourniParGitHub: permet de récupérer du code d'un autre repository sur Github

git push origin master: permet d'envoyer son code local sur Github

git pull origin master: permet de récupérer en local les modifications du repo Github
